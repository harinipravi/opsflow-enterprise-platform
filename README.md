# OpsFlow Enterprise Platform

Enterprise workflow automation and operations management platform built using:

- React
- Vite
- Express.js
- MySQL
- Tailwind CSS

## Features

- Workflow management
- Bulk upload validation
- Analytics dashboard
- Reports module
- Operations console
- Enterprise admin UI

## Tech Stack

Frontend:
- React
- Vite
- Tailwind CSS

Backend:
- Express.js
- Node.js
- MySQL
